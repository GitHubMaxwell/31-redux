## Usage Instructions
* clone fork repo
* install depends
* npm run watch
* enter information and press `Submit` to test CATEGORY_CREATE
* press `Delete` to test CATEGORY_DESTROY
* doubleclick the LI to open update form and enter info and press `Submit` in that form to update the note
